Food Delivery Analysis Project

Analyzing Order Data for Insights

Project Overview

This project analyzes the profit margins, customer behavior, discount effects, and delivery performance of a food delivery service. The dataset includes various attributes such as order value, discount types, profit margins, and delivery times, providing insights into key performance indicators.

Project Goals:

  • Analyze profit margins
  • Segment customers based on spending behavior
  • Evaluate the effect of delivery times on profit
