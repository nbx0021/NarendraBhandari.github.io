Project Overview

AtliQ Hardware, a rapidly expanding consumer electronics company, faced challenges in competing with industry leaders due to their reliance on Excel reports. My mission was to implement an advanced analytics solution using Power BI to provide actionable insights and support informed decision-making across various departments.

Project Goals:

  • User-Centric Report Design
  • Drillable Insights

Skills gained:

  • Power Query (Basic and Advanced Operations)
  • Complex DAX Formulas
  • Data Modeling with 10+ Tables
  • Appropriate Visuals Selection
  • Dashboard Design Principles
  • Using Bookmarks and Filters
  • Power BI Service Deployment
