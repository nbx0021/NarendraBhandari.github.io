This project utilizes machine learning to predict the likelihood of a person developing diabetes. By using health data such as glucose levels, BMI, age, and more, the model is trained to predict whether a person has diabetes or not.

Key Features:

  • Predictive Models: Logistic Regression, Decision Tree, Random Forest, KNN, and SVM
  • Evaluation Metrics: Accuracy, Precision, Recall, F1-Score, AUC-ROC
  • Tools Used: Python, Scikit-learn, Jupyter Notebooks

Insights:

  • Random Forest provides the highest accuracy compared to other models.
  • Logistic Regression is efficient for binary classification tasks.
  • Precision and Recall metrics are key to evaluating model performance.

üìä Model Evaluation

Each model was evaluated using confusion matrices and classification reports. The best performing model was selected based on accuracy, AUC-ROC curve, and other evaluation metrics.

Evaluation Metrics:

  • Logistic Regression: Accuracy = 70%, AUC = 0.81
  • Random Forest: Accuracy = 77%, AUC = 0.82
  • Decision Tree: Accuracy = 68%, AUC = 0.64
  • KNN: Accuracy = 75%, AUC = 0.79
  • SVM: Accuracy = 74%, AUC = 0.80
