Stock News Sentiment Analysis Project

Analyzing Stock Market Sentiment Using Python

Project Overview

This project focuses on analyzing stock market sentiment using news headlines. By scraping stock news data and performing sentiment analysis, the project provides insights into the positive, neutral, and negative sentiment of various stock tickers.

Goals:

  • Scrape stock news headlines using web scraping techniques.
  • Perform sentiment analysis using NLTK's VADER library.
  • Visualize sentiment trends for different stocks.

The project uses Python to extract news headlines from websites like Finviz using BeautifulSoup, classify them as positive, neutral, or negative, and visualize the sentiment trends over time for popular stock tickers like AAPL, MSFT, GOOG, etc.

Steps Involved:

  • Data Scraping: Extract stock news headlines using BeautifulSoup.
  • Sentiment Analysis: Use VADER to determine sentiment scores.
  • Data Visualization: Generate charts to show sentiment trends.
