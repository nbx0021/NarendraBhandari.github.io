Goodcabs Data Analysis Project

Comprehensive Analysis for Operational Insights

Project Overview

This project analyzes trip data for Goodcabs, focusing on key performance metrics such as trip volume variance, repeat passenger rates, and revenue contribution. Insights from this analysis aim to improve customer satisfaction, operational efficiency, and revenue growth.

Project Objectives

  • Analyze city-wise trip performance to identify trends and patterns.
  • Evaluate actual performance against targets for trips and ratings.
  • Identify high and low-demand periods across cities.
  • Assess repeat passenger trends and city contributions to loyalty programs.
  • Provide actionable recommendations to enhance operational efficiency and revenue.

Key Findings

Top and Bottom Cities by Trip Volume Variance

  • Top Cities: Jaipur (9,388), Mysore (2,738), Kochi (1,202)
  • Bottom Cities: Lucknow (-7,701), Vadodara (-5,474), Surat (-2,157)

Repeat Passenger Ratio (RPR%)

  • Top Cities: Surat (42.63%), Lucknow (37.12%), Indore (32.68%)

Average Distance Traveled and Fare

  • Top Cities: Jaipur (30.02 KM, ₹483), Kochi (24.07 KM, ₹335.25)
  • Bottom Cities: Surat (11.00 KM, ₹117.27), Vadodara (11.52 KM, ₹118.27)
